Usage
=====

Please ensure you've completed all :ref:`setup` steps before proceeding.

- In terminal, run:

.. code-block:: bash

    bateman <YOUTUBE_URL>

Or run directly without alias:

.. code-block:: bash

    poetry run python -m app.cli "<YOUTUBE_URL>"
